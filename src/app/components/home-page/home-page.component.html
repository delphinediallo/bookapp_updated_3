Welcome to BookApp!

This app that acts as a social forum for users to create their own virtual bookshelf, post book reviews, take part in discussions with other users, and get the latest news in book publishing.

Examples of this type of application include GoodReads and LibraryThing. This app created for Kennesaw State University course MSIT 6203 Fall 2020.

A project by Stephen Birt, Delphine Diallo, Kathreen Greer, Anna Kurenkov. IT Studio Design class
